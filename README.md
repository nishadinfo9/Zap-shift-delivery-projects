**Zap Shift** is a full-featured courier management application serving all of Bangladesh. Users can send parcels using the platform, track shipments in real-time, and make secure payments via Stripe. The application supports **role-based access** for Admin, Rider, and User, providing specialized dashboards and functionalities for each role.  

This project is a **real-world, production-grade application**, designed to streamline parcel delivery, payment handling, and operational management for courier services.
